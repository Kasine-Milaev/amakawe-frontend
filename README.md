Amakawe — Аниме-каталог в Telegram Mini App
 ## О проекте
Amakawe — это современный аниме-каталог, разработанный как Telegram Mini App. Приложение предоставляет удобный доступ к базе аниме, позволяет искать, фильтровать и получать рекомендации прямо из Telegram.

 ## Основные возможности
 ## Обширный каталог — тысячи аниме с описаниями, оценками и трейлерами

 ## Умный поиск — фильтрация по жанрам, году, типу и статусу

 ## Гибкие темы оформления — 6 уникальных визуальных тем (темная, светлая, сакура, ночь Токио, бамбук, редфор)

 ## Избранное — сохраняйте любимые аниме для быстрого доступа

 ## Рандомный выбор — функция для случайного подбора аниме

 ## Календарь релизов — следите за выходом новых эпизодов

 ## Адаптивный дизайн — корректное отображение на мобильных устройствах и в Telegram

 ## Архитектура
Проект построен на современном стеке технологий:

text
[Telegram Client] ↔ [Telegram Mini App] ↔ [External Anime APIs]
Основные технологии
Frontend: React 18, Vite, React Router DOM

Стилизация: CSS Modules, CSS-переменные для тем

API интеграция: AniList API для данных об аниме

Хостинг: Vercel/Netlify или любой статический хостинг

Контейнеризация: Docker (опционально)

## 📖 Использование
Для пользователей
Найдите бота Amakawe в Telegram

Нажмите на кнопку меню "Смотреть аниме"

Используйте поиск или просматривайте популярные аниме

Добавляйте аниме в избранное для быстрого доступа

API интеграции
Основной источник данных
Проект использует AniList API для получения:

Списков аниме

Детальной информации

Жанров и студий

Рейтингов и отзывов
